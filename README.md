## Building the site

### Prerequisites
- Hugo (extended version): https://gohugo.io/getting-started/installing/
- Node.js: https://nodejs.org/en/download/

### Build
1. Clone the repository
2. Run `hugo server -D`
3. Open http://localhost:1313/ in your browser
